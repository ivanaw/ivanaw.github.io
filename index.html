PROJECTS

Flock

My team and I built a module to accurately track and report GPS locations of a bike fleet at UMD using the LinkIt One Iot that periodically sends data to Firebase database via GSM. We also built an intuitive web application using Node.Js to view data in realtime and allows users to control bike setting.

SnapLingua

My team and I created this project at PennApps XVI (2017). This project allows users to take learn a new language by taking pictures of the object and the application will translate that picture from English to your desired language. We coded the project in swift and used the Google Vision API and Google Translate api to create this application.

Lullabyte

My team and I worked on this project at Cal Hacks 4.0 (2017). We noticed how parents and children are not interacting with one another because of our screens. We wanted to change that and let parents and children use their screen to interact together. This allows the parent and child to create a story togther and pictures and gifs will appear making their story into "reality". We attempted to do code it using Expo and used the Google Speech to text API and Google Image API.
